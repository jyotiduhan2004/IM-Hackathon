## § The bombshell finding — V9-U1 schema is misfit

The 303-topic archetype survey (`topic-page-structure-archetypes-2026-04-18.md`) did the analysis V11's design assumed unnecessary:

- **0 of 303 topic pages** satisfy V9-U1's "5 of 8 canonical H2s" bar.
- **0 topics** carry `## Current state`, `## Why it matters`, `## Key decisions`, or `## Recent changes`.
- **218 of 303 (72%)** converge on a "design-doc" shape never explicitly taught: Overview / Background / Problem / Solution / Implementation / Results / Way forward.
- **283 of 304 (97.4%)** carry `## Related`.
- Topic openers: 111 `## Overview`, 52 `## Business Objective`, 21 `## TL;DR`, 1 `## Summary` (V9-U1's lead).

This run's 7 pages match the corpus exactly — 5/7 design-doc, 1 thread-narrative, 1 design-doc+system-lite. **V11-U7's enforcement work is targeting the wrong vocabulary**. The reviewer escalation NOT firing is the absence of a wrong action. If U7 had been wired hard against V9-U1, all 7 pages would be blocked and rewritten as `## Summary/Current state/Why it matters` shells wrapping content that already fits a different (better) shape. The V11-U7 critique-warning is fine as a diagnostic; the reviewer-escalation step should NOT be wired against V9-U1. Replace the schema instead.

## § Recommended V12 units

### V12-U1 — REPLACE `SUGGESTED_SECTIONS["topic"]` (HEADLINE)

Highest-leverage change in V12. Adopt the 7-slot schema the archetype audit derived from organic structure:

```
topic (default):
  - Summary             # or TL;DR — both acceptable
  - Background          # why this exists, problem framing
  - What changed        # or Implementation / Solution
  - Metrics / Impact    # expected + observed
  - Current status      # v9-U1's 'Current state' — keep slot
  - Open questions      # keep; 12 organic topics use it
  - Related             # 97.4% of pages already have it
```

File changes:
- `src/compile/section_shapes.py:22` — replace `SUGGESTED_SECTIONS["topic"]` list.
- `src/compile/critique.py` — update `_check_suggested_h2_sections` substring-match list to allow synonyms (`Summary|TL;DR|Overview`, `Background`, `What changed|Implementation|Solution`, `Metrics|Impact|Results`, `Current status|Current state`, `Open questions`, `Related`).
- `src/compile/reviewer.py` — update `REVIEWER_SYSTEM_PROMPT` `structure_mismatch` / `filing_cabinet` text to reference the new shape.
- `src/compile/prompts.py` — update topic-shape guidance.

Risk: low. Existing pages don't need backfill — already comply at ~70%+. Reviewer false-positive rate on `missing_suggested_h2s` should drop from 100% (current) to ~30%.
